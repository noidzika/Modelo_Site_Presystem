## 🎯 Sobre o projeto

O projeto é um modelo de site institucional para empresa de tecnologia e eletrônica, desenvolvido do zero com foco em transmitir credibilidade, modernidade e expertise técnica.

O projeto demonstra na prática como é possível construir uma experiência de **Single Page Application (SPA)** utilizando apenas **HTML, CSS e JavaScript puro**, sem depender de nenhum framework.

---

## 🧠 Destaques técnicos

### Navegação SPA com JavaScript puro
A maior entrega técnica desse projeto é a arquitetura de navegação. Em vez de múltiplas páginas HTML, **todo o conteúdo vive em um único arquivo**, e o JavaScript gerencia qual seção é exibida conforme o link clicado na navbar:

- Cada seção é um bloco independente no HTML
- O JS escuta os cliques do menu e alterna a visibilidade das seções
- A URL não muda, a página não recarrega — comportamento idêntico ao de frameworks modernos
- A navbar destaca o item ativo dinamicamente com CSS controlado por JS

Essa abordagem evidencia **domínio real de JavaScript** — sem atalhos, sem bibliotecas.
